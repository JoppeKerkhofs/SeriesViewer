# Series Viewer

Series Viewer is a simple Electron application that allows you to watch your favorite TV series in a convenient and organized manner. It provides a user-friendly interface for selecting and watching episodes of your series. You can easily navigate between seasons and episodes, customize settings, and enjoy a seamless viewing experience.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)

## Installation

1. Clone the repository or download the project files.
2. Run `npm install` to install the required dependencies.

## Usage

1. Run the application in develop mode using `npm start` or `npm run dev`.
2. Build the application using `npm run make`.
